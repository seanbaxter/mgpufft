MGPU FFT 0.01

MGPU FFT is component library for Fast Fourier Transforms on the CUDA platform. The intended use is for accelerating small-image convolutions using the convolution theorem.

In this 0.01 version, only 1D FFTs for power-of-two sizes between 4 and 1024 are included. They are benchmarked against CUFFT's equivalent cudaPlanMany transforms.
